# CyberRakshak Requirements

## Problem Statement

People in rural and semi-urban areas are increasingly using UPI, smartphones, and digital services but lack cyber security awareness, making them vulnerable to scams, phishing attacks, OTP frauds, and fake calls.

## Objectives

1. Improve cyber awareness.
2. Educate users about digital safety.
3. Provide AI-based guidance.
4. Help cyber crime victims.
5. Reduce cyber fraud incidents.

## Target Users

* Students
* Farmers
* Shopkeepers
* Elderly People

## Features

1. Learning Center
2. AI Chatbot
3. Emergency Help
4. Cyber Crime Solutions
5. Feedback System
6. Multi-Language Support
